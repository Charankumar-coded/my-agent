# AI Agent Demo

This project is a single-page demo for an AI-style agent workflow. It now runs in a normal browser demo mode without requiring an API key.

## What changed
- Removed the API-key input requirement from the interface.
- Switched the agent flow to built-in demo responses so the page works immediately.
- Kept the Think → Plan → Execute → Deliver experience intact for preview and testing.

## How to use it
1. Open index.html in a browser, or serve the folder with a simple local server.
2. Enter a task in the input box.
3. Click Run Agent to see the four-step workflow.

## Notes
- This demo is intentionally self-contained and does not send any secret key.
- If you later want live AI responses, you can connect the page to a backend or API provider.
